# sikopet ERP Design System (UI Kit)

> Design Style: Modern, Minimal, Enterprise Dashboard, Soft UI

---

# Design Principles

- Clean and uncluttered
- Professional ERP appearance
- Mobile-first responsive
- Soft rounded corners
- Spacious layout
- High readability
- Minimal accent colors
- Information hierarchy through spacing instead of borders
- Consistent 8pt spacing system
